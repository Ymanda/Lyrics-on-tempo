🎵 Caleur de paroles

écris tes paroles sur un air connu — l'outil compte les syllabes en direct. Libre, sans compte.
📖 Mode d'emploi
1. Choisis un air dans le menu déroulant (ou « Paroles d'origine » pour partir du texte connu).
2. Écris tes paroles, un vers par ligne. À droite, chaque vers passe au vert quand le nombre de syllabes tombe juste — orange = mélisme toléré, rouge = trop, bleu = pas assez.
3. « Écouter l'air » joue la mélodie. Si le français est ambigu, ajuste avec les marqueurs collés au mot : + ajoute une syllabe, = en retire, _ force une coupure.
4. Quand ça te plaît : ⬇ MIDI / ⬇ Partition (ouvre dans MuseScore) / ⬇ Paroles, ou 🔗 Lien pour partager. Ton texte est gardé automatiquement sur cet appareil.
